## Contributing

We welcome contributions of all kinds!

### Code Contributions

- **New Tools**: Add MCP tools for Windows automation
- **Bug Fixes**: Fix issues in existing tools
- **Performance**: Optimize code for speed and efficiency
- **Refactoring**: Improve code structure and maintainability

### Non-Code Contributions

- **Documentation**: Improve guides, tutorials, and API docs
- **Testing**: Add test cases or improve coverage
- **Bug Reports**: Report detailed, reproducible issues
- **Feature Ideas**: Suggest new features and improvements
- **Community Support**: Help others on Discord or Discussions
- **Translations**: Help translate documentation (coming soon)

## Code of Conduct

By participating in the Windows-MCP community, you agree to:

- **Be respectful and considerate** in all communications
- **Welcome newcomers** and help them get started
- **Accept constructive criticism** gracefully
- **Focus on what's best** for the community and project
- **Maintain a harassment-free** environment for everyone

We're committed to providing a welcoming and inclusive community for all participants.
