## What is Windows-MCP?

Windows-MCP is a lightweight, open-source MCP (Model Context Protocol) server that bridges AI agents with the Windows operating system. It enables seamless desktop automation through 17 powerful tools for UI interaction, file management, system control, and more.

<CardGroup cols={2}>
  <Card title="Works with Any LLM" icon="brain">
    No vision models required. Works with Claude, GPT, Gemini, and any LLM that supports MCP.
  </Card>
  <Card title="17 Automation Tools" icon="wrench">
    Complete toolkit for UI automation, file operations, system commands, and browser control.
  </Card>
  <Card title="Real-Time Interaction" icon="bolt">
    Lightning-fast action latency (0.2-0.9 seconds) for responsive automation workflows.
  </Card>
  <Card title="Local & Remote" icon="cloud">
    Deploy locally or use cloud-hosted Windows VMs via windowsmcp.io for remote automation.
  </Card>
</CardGroup>

## Supported MCP Clients

Windows-MCP works with all major MCP clients:

- **Claude Desktop** - Anthropic's desktop app
- **Perplexity Desktop** - AI search and automation
- **Gemini CLI** - Google's command-line interface
- **Qwen Code** - Alibaba's coding assistant
- **Codex CLI** - OpenAI's development tool
